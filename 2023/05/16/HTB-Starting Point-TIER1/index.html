HTB-Appointment

TASK1

What does the acronym SQL stand for?

问题：SQL 缩写代表什么？

答案：Structured Query Language

TASK2

What is one of the most common type of SQL vulnerabilities?

问题：最常见的 SQL 漏洞类型之一是什么？

答案：SQL injection（SQL注入）

TASK3

What is the 2021 OWASP Top 10 classification for this vulnerability?

问题：此漏洞的 2021 年 OWASP Top 10 分类是什么？

答案：A03:2021-Injection

TASK4

What does Nmap report as the service and version that are running on port 80 of the target?

问题：Nmap 报告在目标的 80 端口上运行的服务和版本是什么？

答案：Apache httpd 2.4.38 ((Debian))

TASK5

What is the standard port used for the HTTPS protocol?

问题：HTTPS 协议使用的标准端口是什么？

答案：443

TASK6

What is a folder called in web-application terminology?

问题：Web 应用程序术语中的文件夹是什么？

答案：directory

TASK7

What is the HTTP response code is given for ‘Not Found’ errors?

问题：“未找到”错误的 HTTP 响应代码是什么？

答案：404

TASK8

Gobuster is one tool used to brute force directories on a webserver. What switch do we use with Gobuster to specify we’re looking to discover directories, and not subdomains?

问题：Gobuster 是一种用于暴力破解网络服务器上目录的工具。我们在 Gobuster 中使用什么开关来指定我们要发现目录，而不是子域？

答案：dir

TASK9

What single character can be used to comment out the rest of a line in MySQL?

问题：MySQL 中可以使用什么单个字符来注释掉一行的其余部分？

答案：#

TASK10

If user input is not handled carefully, it could be interpreted as a comment. Use a comment to login as admin without knowing the password. What is the first word on the webpage returned?

问题：如果不仔细处理用户输入，它可能会被解释为注释。使用注释以管理员身份登录，无需知道密码。返回的网页第一个单词是什么？

答案：Congratulations

问题：哪个用户名允许我们在不提供密码的情况下登录此 MariaDB 实例？

HTB-Responder

TASK1

When visiting the web service using the IP address, what is the domain that we are being redirected to?

问题：当使用 IP 地址访问 Web 服务时，我们被重定向到的域是什么？

答案：unika.htb

先用nmap扫描端口，发现3个开放端口，其中5985端口为WinRM（Windows Remote Management），Windows 远程管理是 Windows 原生的内置远程管理协议。

WinRM 允许用户：
→ 与主机进行远程通信
→ 在非本地但可通过网络访问的系统上远程执行命令。
→ 从远程位置监控、管理和配置服务器、操作系统和客户端计算机。
这意味着如果我们可以找到某个用户的凭据（通常是用户名和密码）具有远程管理权限，我们有可能在主机上获得shell。

重定向到的unika.htb不能访问到，我们需要修改本地的hosts文件（实现主机名到IP的映射），Linux系统在 /etc/hosts ，将下面的内容写入该文件，我们就可以访问到该网站了

TASK6

What does NTLM stand for?

问题：NTLM 代表什么？

答案：New Technology Lan Manager

NTLM(New Technology LAN Manager)身份验证协议是微软用于Windows身份验证的主要协议之一。早期SMB协议以明文口令的形式在网络上传输，因此产生了安全性问题。后来出现了LM(LAN Manager)身份验证协议，它是如此的简单以至于很容易被破解。后来微软提出了NTLM身份验证协议，以及更新的NTLM V2版本。NTLM协议既可以为工作组中的机器提供身份验证，也可以用于域环境身份验证。NTLM协议可以为SMB、HTTP、LDAP、SMTP等上层微软应用提供身份认证。
