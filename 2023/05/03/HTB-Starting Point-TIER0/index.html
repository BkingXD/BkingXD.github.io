HTB-Starting Point-TIER0

HTB-Meow

TASK1

What does the acronym VM stand for?

问题：VM的缩写代表什么？

答案：Virtual Machine（虚拟机）

TASK2

What tool do we use to interact with the operating system in order to issue commands via the command line, such as the one to start our VPN connection? It’s also known as a console or shell.

问题：我们使用什么工具与操作系统交互，以便通过命令行发出命令，例如启动 VPN 连接的工具？它也称为控制台或外壳。

答案：terminal（终端）

TASK3

What service do we use to form our VPN connection into HTB labs?

问题：我们使用什么服务来建立与 HTB 实验室的 VPN 连接？

答案：openvpn

TASK4

What tool do we use to test our connection to the target with an ICMP echo request?

问题：我们使用什么工具通过 ICMP 回显请求来测试与目标的连接？

答案：ping

TASK5

What is the name of the most common tool for finding open ports on a target?

问题：用于查找目标上开放端口的最常用工具的名称是什么？

答案：nmap

TASK6

What service do we identify on port 23/tcp during our scans?

问题：在扫描过程中，我们在端口 23/tcp 上识别出哪些服务？

答案：telnet

TASK7

What username is able to log into the target over telnet with a blank password?

问题：什么用户名能够使用空白密码通过 telnet 登录目标？

答案：root

SUBMIT FLAG

首先进行端口扫描，发现23端口开放，telnet服务

我们直接telnet远程登录，根据上面的task7，root账户不需要密码，远程登录后在当前目录下读到flag

HTB-Fawn

TASK1

What does the 3-letter acronym FTP stand for?

问题：3 个字母的缩写 FTP 代表什么？

答案：file transfer protocol

TASK2

Which port does the FTP service listen on usually?

问题：FTP服务通常监听哪个端口？

答案：21

TASK3

What acronym is used for the secure version of FTP?

问题：FTP 的安全版本使用什么缩写词？

答案：SFTP

TASK4

What is the command we can use to send an ICMP echo request to test our connection to the target?

问题：我们可以使用什么命令发送 ICMP 回显请求来测试与目标的连接？

答案：ping

TASK5

From your scans, what version is FTP running on the target?

问题：根据您的扫描，目标上运行的 FTP 版本是什么？

答案：vsftpd 3.0.3

TASK6

From your scans, what OS type is running on the target?

问题：根据您的扫描，目标上运行的操作系统类型是什么？

答案：Unix

TASK7

What is the command we need to run in order to display the ‘ftp’ client help menu?

问题：为了显示“ftp”客户端帮助菜单，我们需要运行什么命令？

答案：ftp -h

TASK8

What is username that is used over FTP when you want to log in without having an account?

问题：当您想在没有帐户的情况下登录时，通过 FTP 使用的用户名是什么？

答案：anonymous

TASK9

What is the response code we get for the FTP message ‘Login successful’?

问题：我们收到的 FTP 消息“登录成功”的响应代码是什么？

答案：230

TASK10

There are a couple of commands we can use to list the files and directories available on the FTP server. One is dir. What is the other that is a common way to list files on a Linux system.

问题：我们可以使用几个命令来列出 FTP 服务器上可用的文件和目录。一个是目录。另一种是在 Linux 系统上列出文件的常见方法。

答案：ls

TASK11

What is the command used to download the file we found on the FTP server?

问题：用于下载我们在 FTP 服务器上找到的文件的命令是什么？

答案：get

SUBMIT FLAG

get下载得到flag.txt，查看就可以得到flag

HTB-Dancing

TASK1

What does the 3-letter acronym SMB stand for?

问题：3 个字母的缩写 SMB 代表什么？

答案：Server Message Block

TASK2

What port does SMB use to operate at?

问题：SMB 使用什么端口进行操作？

答案：445

TASK3

What is the service name for port 445 that came up in our Nmap scan?

问题：我们的 Nmap 扫描中出现的端口 445 的服务名称是什么？

答案：microsoft-ds

TASK4

What is the ‘flag’ or ‘switch’ that we can use with the smbclient utility to ‘list’ the available shares on Dancing?

问题：我们可以与 smbclient 实用程序一起使用来“列出”Dancing 上的可用共享的“标志”或“开关”是什么？

答案：-L（–list）

TASK5

How many shares are there on Dancing?

问题：Dancing有多少shares？

答案：4

TASK6

What is the name of the share we are able to access in the end with a blank password?

问题：我们最终能够使用空白密码访问的共享名称是什么？

答案：WorkShares

不尝试连接 IPC$ 是因为它没有价值，它不能像任何常规目录一样可浏览，并且不包含我们在这一学习阶段可以使用的任何文件。

TASK7

What is the command we can use within the SMB shell to download the files we find?

问题：我们可以在 SMB shell 中使用什么命令来下载我们找到的文件？

答案：get

SUBMIT FLAG

flag在 James.P 文件夹里面，get命令下载即可

HTB-Redeemer

TASK1

Which TCP port is open on the machine?

问题：机器上打开了哪个 TCP 端口？

答案：6379

使用nmap进行端口扫描，使用参数 -p- 可扫描全端口1~65535

TASK2

Which service is running on the port that is open on the machine?

问题：计算机上打开的端口上正在运行哪个服务？

答案：redis

TASK3

What type of database is Redis? Choose from the following options: (i) In-memory Database, (ii) Traditional Database

问题：Redis 是什么类型的数据库？从以下选项中进行选择：(i) 内存数据库，(ii) 传统数据库

答案：In-memory Database

TASK4

Which command-line utility is used to interact with the Redis server? Enter the program name you would enter into the terminal without any arguments.

问题：哪个命令行实用程序用于与 Redis 服务器交互？输入您要在终端中输入的程序名称，不带任何参数。

答案：redis-cli

TASK5

Which flag is used with the Redis command-line utility to specify the hostname?

问题：Redis 命令行实用程序使用哪个标志来指定主机名？

答案：-h

TASK6

Once connected to a Redis server, which command is used to obtain the information and statistics about the Redis server?

问题：连接到Redis服务器后，使用哪个命令来获取Redis服务器的信息和统计信息？

答案：info

TASK7

What is the version of the Redis server being used on the target machine?

问题：目标机器上使用的Redis服务器版本是什么？

答案：5.0.7

TASK8

Which command is used to select the desired database in Redis?

问题：使用哪个命令在 Redis 中选择所需的数据库？

答案：select

Keyspace提供了每个数据库主字典的统计信息。 统计数据包括键的数量，以及过期的键的数量。

TASK9

How many keys are present inside the database with index 0?

问题：索引为0的数据库中有多少个键？

答案：4

TASK10

Which command is used to obtain all the keys in a database?

问题：哪个命令用于获取数据库中的所有键？

答案：keys *

SUBMIT FLAG

获取到所有键名之后，用get命令获取键名所对应的键值。
